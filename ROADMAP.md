# Exchange Univ Map — Roadmap

## 비전

서강대학교 교환학생 지원자가 지도 하나로 모든 정보를 탐색하고, AI 어드바이저에게 맞춤 추천까지 받을 수 있는 원스톱 플랫폼.

---

## Phase 1: AI 교환학생 어드바이저 MVP `🔨 진행 중`

**목표:** Claude API 기반 자연어 검색 + 맞춤 진단 기능 추가

**기간:** 2026년 5월

| 항목 | 상태 |
|------|------|
| 기본 지도 + 마커 시스템 | ✅ 완료 |
| 대륙 필터 버튼 | ✅ 완료 |
| 버그 수정 (마커 중복, 경로 오류, 로그) | ✅ 완료 |
| AI 추천 백엔드 API (`/api/recommend`) | 🔨 진행 중 |
| AI 어드바이저 UI 사이드 패널 | 🔨 진행 중 |
| 자유 검색 탭 (자연어 입력) | 🔨 진행 중 |
| 맞춤 진단 탭 (GPA·언어·지역 입력) | 🔨 진행 중 |
| GitHub 배포 | 🔨 진행 중 |

## Phase 2: 서강대 국제처 RAG 어시스턴트 `📋 계획`

**목표:** 서강대 공식 문서 기반 Q&A 챗봇 통합

| 항목 | 상태 |
|------|------|
| 국제처 홈페이지 공식 문서 크롤링 | 📋 계획 |
| 문서 청킹 + 임베딩 파이프라인 | 📋 계획 |
| `/api/ask` RAG 엔드포인트 구현 | 📋 계획 |
| Q&A 챗봇 UI 통합 | 📋 계획 |

**예상 Q&A 커버리지:**
- 교환학생 지원 절차 및 서류
- 학점 인정 기준
- 파견 장학금 정보
- 지원 마감일 안내
